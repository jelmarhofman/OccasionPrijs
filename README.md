# OccasionPrijs

Occasion Prijsmodel – een standalone webapplicatie voor het berekenen van de marktwaarde van gebruikte auto's.

## 🌐 Website bekijken

De website is beschikbaar via GitHub Pages:

**👉 https://jelmarhofman.github.io/OccasionPrijs/**

_(Klik op bovenstaande link om het prijsmodel te openen – ook toegankelijk voor de docent.)_

## 📋 Functionaliteiten

- Voertuiggegevens invoeren (merk, bouwjaar, km-stand, brandstof, etc.)
- Staat van het voertuig beoordelen (carrosserie, interieur, onderhoud, APK)
- Extra opties selecteren
- Automatische prijsberekening met bandbreedte (±7%)
- Gedetailleerde prijsopbouw per factor
- Grafiek met verwachte waardedaling

## 🚀 Lokaal openen

Download of kloon dit project en open `index.html` direct in een webbrowser. Geen server nodig – de applicatie is volledig standalone.
